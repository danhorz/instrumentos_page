Juego 1

Hermanos: El programa pedirá al usuario que ingrese la cantidad de hermanos. Este número determinará el límite de intentos. A continuación, tienes tantos intentos como el número que hayas indicado(Usa el bucle while). El programa te pedirá que ingreses los nombres de cada uno de tus hermanos, uno por uno, y los agregará en un arreglo. Al finalizar, usando console.log() el programa mostrará el arreglo con todos los nombres de los hermanos que hayas ingresado. 


                        
                            //Eje1
                            function registroDeHermanos()
                            {
                                const cantidadDeHermanos=prompt("Ingrese la cantidad de hermanos:")

                                const listaHermanos=[];
                                let contador=0;
                                while(cantidadDeHermanos>contador)
                                {                               let nombreDeHermano= prompt("Ingresa el hermano"+contador);
                                    listaHermanos.push(nombreDeHermano);
                                    contador++;
                                }   
                                for(list of listaHermanos)
                                {
                                    console.log(list);
                                }
                            }

                            //Eje2

                            function colorAleatorioHex() {
                                let caracteres = "0123456789ABCDEF";
                                let color = "#";
                                for (let i = 0; i < 6; i++) {
                                  color += caracteres[Math.floor(Math.random() * 16)];
                                }
                                return color;
                              }

                            function elegirOpcionAleatoria() {
                                const opciones = ["piedra", "papel", "tijera"];
                                const indiceAleatorio = Math.floor(Math.random() * opciones.length);
                                return opciones[indiceAleatorio];
                              }

Juego 4

Calculadora de Edad

  • Reciba el año de nacimiento de una persona
  • Calcule y devuelva su edad actual
  • Muestre en consola: "Tienes X años"

    • Ejemplo: calcularEdad(1990) // Debería mostrar "Tienes 34 años"

    
                        
                            function ej4()
                            {
                                const añoIngresado=Number(prompt("Ingrese su año de nacimiento de una persona"));
                                const añoActual=new Date().getFullYear();
                                const edad=añoActual-añoIngresado;
                                alert("Tienes "+edad+" años");
                            }

  • Retorne el resultado con un mensaje
